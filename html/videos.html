DVO Toolbar and GunDB

NOTE: The extension now uses both GunDB and Solid. So, instead of writing the comments to GunDB, it only stores a comment ID. The comment itself is stored in the commenter's POD - public/DVO/comments/

GunDB is a P2P, graph database engine, written entirely in Javascript. By default, data is stored in your localStorage, and is replicated to other peers using your application. GunDB is eventually consistant, and uses a sophisticated conflict resolution algorythm. The syntax is similar to Firebase.
