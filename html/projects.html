Projects

  • DVO Profile Manager: your data is stored in a Personal Online Datastore (POD). For more information see Solid - a project that is being lead by Tim Berner's Lee - the founder of the World Wide Web. You can host this POD with whoever you choose. However, given that the underlying technology (Solid) is still in its infancy, there are currently only two providers to choose from.
  • DVO Toolbar: using this extension you can like, dislike and comment on any URL. Please go to the Videos section to see it in action.
  • DVO Search: the search results will work alongside Google, but instead of using algorythms to determine the quality and relevancy of the content, it be determined by the users.
  • Devolution: a governance platform where users create services, vote for services, and finance services. However, as it stands, most of this will be done using the DVO Toolbar. Please go to the Videos section to see it in action.
  • DVO Modular Housing: a free-standing, flatpack, modular housing system built using aluminium profile. This enables people to easily build and maintain their own homes. Please go to the Videos to see the prototype.
  • DVO Lottery: a decentralized lottery where the proceeds are returned to the users in the form of credits, which they can use to finance services, either using their toolbar, or on the Devolution platform.
  • DVO Adverts: as above, the proceeds are returned to the users in the form of credits, which they can use to finance services.
  • DVO Directory: a directory of both business and personal profiles, with an advanced search facility.
  • DVO Dating: a free online dating service that can be used by anyone, anywhere in the world.
